# MMORPG 乘区伤害计算器

一个MMORPG游戏伤害计算模拟器，支持复杂的乘区计算、多种职业系数计算、完整的历史记录管理系统和全新的幸运一击伤害计算功能。

## 📁 文件结构

```
Star app/
├── index.html     # 主页面结构和所有弹窗定义 
├── styles.css     # 完整样式文件和响应式设计 
├── script.js      # 主要计算逻辑和历史记录管理 
├── modals.js      # 弹窗功能和智能计算器实现 
├── debug.js       # 调试辅助脚本 
├── diagnose.js    # 系统诊断和功能检测脚本
└── README.md      # 项目说明文档
```

## 🚀 使用方法

1. 双击 `index.html` 文件在浏览器中打开
2. 在左侧分组卡片中输入各项数值参数
3. 右侧实时显示计算结果和提升百分比
4. 点击带有❓图标的参数旁边可以打开对应的专业计算器
5. 使用💾保存记录按钮手动保存当前配置
6. 使用📊历史记录按钮查看、应用和管理历史记录

### 📊 智能计算器系统

#### 1. 攻击力计算器 ⚔️
- **组成部分**: 固定值 + 主属性转化
- **计算公式**: `攻击力区 = 固定值 + 主属性转化`
- **主属性转化子计算器**: 点击❓进入专门的主属性转化计算
- **实时联动**: 修改任一组件立即更新攻击力区结果

#### 2. 主属性转化计算器
- **输入参数**: 主属性、G系数
- **计算公式**: `主属性转化 = 主属性 × G`
- **智能应用**: 计算结果可直接应用到攻击力计算器
- **独立弹窗**: 专用计算器界面，支持精确配置

#### 3. 元素伤害乘区计算器 🔥
- **默认项目**: 固定值(%)、精通转化(%)
- **精通转化子计算器**: 点击❓进入专门的精通转化计算
- **动态项目**: 可添加无限个自定义元素伤害项目并命名
- **计算公式**: `元素伤害乘区 = 1 + 固定值/100 + 精通转化/100 + 其他项目/100`
- **百分比输入**: 所有输入值按百分比处理，界面更直观

#### 4. 精通转化计算器
- **输入参数**: 固定值、获取量、Y系数、T系数（精通增效）
- **精通乘区公式**: `精通乘区 = 固定值 + (获取量 / (获取量 + 50000))`
- **精通增效公式**: `精通增效 = T × (1 + 精通乘区%)`
- **开关控制**: 支持精通增效开启/关闭，关闭时 `精通增效 = 1`
- **精通转化公式**: `精通转化 = [固定值 + (获取量 / (获取量 + 50000))] × Y × 精通增效 × 100`
- **百分比结果**: 自动转换为百分比形式供元素伤害使用

#### 5. 幸运概率计算器 🍀
- **输入参数**: 固定值、获取量
- **计算公式**: `幸运概率 = 固定值 + (获取量 / (获取量 + 50000))`
- **智能联动**: 计算结果自动传递给幸运倍率计算器
- **实时更新**: 修改参数立即更新幸运概率值

#### 6. 幸运倍率计算器 ⭐
- **输入参数**: 基础倍率(可自定义，默认40%)、幸运概率、X系数
- **计算公式**: `幸运倍率 = 基础倍率 + 幸运概率/X`
- **灵活配置**: 基础倍率可自定义修改，不限于固定的40%
- **系数调节**: X系数可调节幸运概率对倍率的影响程度

#### 7. 爆伤期望计算器 💥
- **输入参数**: 暴击率(%)和暴击伤害(%)
- **计算公式**: `期望暴击伤害 = 1 + (暴击率/100) × (暴击伤害/100 - 1)`
- **可折叠公式**: 点击📝计算公式按钮展开/折叠详细说明

#### 8. 全能增幅计算器
- **输入参数**: 固定值、获取量、Q系数
- **全能乘区公式**: `1 + 固定值 + (获取量 / (获取量 + 28000))`
- **全能增幅公式**: `1 + [固定值 + (获取量 / (获取量 + 28000))] × Q`
- **双重计算**: 同时显示全能乘区和全能增幅乘区结果

#### 9. 雷印计算器 ⚡
- **输入参数**: 雷印个数和每个雷印的百分比  
- **计算公式**: `雷印值 = 个数 × (百分比/100)` 
- **独立弹窗**: 专用计算器界面，支持精确配置

#### 10. 精通增幅计算器
- **输入参数**: 固定值、获取量、J系数
- **精通乘区公式**: `精通乘区 = 固定值 + (获取量 / (获取量 + 50000))`
- **精通增幅公式**: `精通增幅 = 1 + [固定值 + (获取量 / (获取量 + 50000))] × J`

#### 11. 增伤计算器
- **基础参数**: 首领增伤百分比、近/远距离加成百分比
- **动态功能**: 可添加无限个自定义增伤项目并命名
- **计算公式**: `1 + 首领增伤/100 + 距离加成/100 + 其他增伤项目/100`
- **统一界面**: 与元素伤害计算器采用相同的添加项目设计
- **灵活管理**: 支持动态添加和删除增伤项目

#### 12. 系数计算器 - 职业专精系统

**雷影剑士** ⚡
- **系数来源**: 系数 M = 雷印 + 精通增幅，系数 N = 雷印 + 精通增幅
- **智能计算**: 自动计算总系数 (雷印 + 精通增幅)
- **灵活分配**: 可将总计复制到M、N或同时分配
- **手动调整**: 支持手动输入M、N系数值
- **实时预览**: 显示当前雷印值、精通值和总计

**其他职业**
- 预留接口，暂时需要手动输入系数值
- 可扩展其他职业的专门计算公式
- 提供系数重置功能：可一键将系数M、N恢复为默认值1

#### 13. 减伤区计算器 🛡️
- **输入参数**: 防御值、F系数  
- **防御区公式**: `防御区 = 防御值 / (防御值 + 1684)`
- **减伤区公式**: `减伤区 = 1 - 防御值 / (防御值 + 1684) × F`
- **双重计算**: 同时显示防御区和减伤区结果
- **自定义系数**: F系数可自定义，提供灵活的减伤计算

### 🍀 幸运一击系统 - 全新特性

#### 核心概念
- **幸运一击**: 独立于减伤区影响的额外伤害来源
- **计算位置**: 在核心伤害计算中，与基础伤害和附加伤害并列
- **乘区独立**: 幸运一击伤害不受减伤区影响，但受其他乘区影响

#### 计算机制
- **幸运伤害公式**: `(攻击力+精炼攻击+元素攻击) × 幸运概率 × 幸运倍率`
- **完整伤害公式**: `[基础伤害 + 附加伤害 + 幸运伤害 + 固定值] × 所有乘区`
- **计算顺序**: 先计算幸运概率，再计算幸运倍率

#### 策略价值
- **伤害提升**: 提供额外的伤害来源，增加总伤害输出
- **属性平衡**: 合理分配幸运属性与其他属性的投入比例
- **系数优化**: 通过调节X系数优化幸运概率到倍率的转换效率

## 🎯 详细使用案例

### 基础伤害计算
1. 在分组卡片中输入角色基础属性
2. 查看实时计算的最终伤害和详细公式
3. 点击公式展开按钮查看详细计算过程
4. 设置基准伤害，分析提升百分比

### 幸运一击配置流程 🍀
1. 点击幸运概率旁的❓图标打开幸运概率计算器
2. 输入固定值和获取量，查看计算结果
3. 应用幸运概率到主界面
4. 点击幸运倍率旁的❓图标打开幸运倍率计算器
5. 调整基础倍率（默认40%可自定义）和X系数
6. 查看幸运概率自动填入的值，调整参数
7. 应用幸运倍率到主界面
8. 观察幸运一击对总伤害的影响

### 攻击力区精确配置流程
1. 点击攻击力旁的❓图标打开攻击力计算器
2. 配置固定值攻击力
3. 点击主属性转化❓ → 输入主属性和G系数 → 应用主属性转化结果
4. 在攻击力计算器中查看总攻击力区
5. 应用到主界面进行最终伤害计算

### 元素伤害乘区配置流程
1. 点击元素伤害旁的❓图标打开元素伤害计算器
2. 配置固定值百分比
3. 点击精通转化❓ → 输入精通固定值、获取量、Y系数、T系数
4. 根据需要切换“精通增效开关”（关闭时精通增效=1）
5. 应用精通转化结果到元素伤害计算器
6. 根据需要添加其他元素伤害项目
7. 查看总元素伤害乘区并应用到主界面

### 雷影剑士专精计算流程
1. 点击系数M/N旁的❓图标打开系数计算器
2. 选择"雷影剑士"职业
3. 配置雷印: 点击雷印❓ → 输入个数和百分比 → 应用结果
4. 配置精通: 点击精通❓ → 输入固定值、获取量、J系数 → 应用结果  
5. 在系数计算器中查看总计，选择分配方式
6. 应用到主界面进行最终伤害计算

### 历史记录管理工作流
